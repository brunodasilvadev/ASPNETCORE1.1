# ASPNETCORE1.1
This project are a simple CRUD with the objective to learn.

Developed with:

- ASP.NET MVC Core
- C# 
- DDD pattern
- Entity Framework Core
- Code First
- Dapper
- Dependency Injection
- FluentValidation 
- CQRS (Command Query Responsibility Segregation)
- Claims Policy
- Decoupling Asp.Net Identity
- API REST
